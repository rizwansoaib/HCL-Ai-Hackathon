## Whatsapp Sentiment Analysis
* Trainig Dataset: Movie reveiw nltk 
* Classifier: NaivBayes
* Testing Dataset: User Whatsapp chat exported text format
### Export Whatsapp Chat
1. Open Chat Click Option (Three dot)
2. Tap on more
3. Click Export chat
4. Without Media 
5. Now use *.txt file
